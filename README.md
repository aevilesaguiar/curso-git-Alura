# Comandos Git
 
##Iniciando repositório

git init

##Clonamos o repositório

git clone <url>

##Adicionamos o arquivo para commitar

	git add .
 
##Fazer o commit

	git commit -m "Texto que identifique por que fez o commit"
  
##Subir o repositório

  git push origin master
	


	GIT CLONE
